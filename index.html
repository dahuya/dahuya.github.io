戒掉抖音等信息流应用的好方法

发表于 2025-01-05

有很多人沉迷于于抖音快手，小红书等平台，这些平台会不停的推荐个人感兴趣的内容，总是容易使人沉迷，戒不掉。在这里教大家一个方法，可以使人不再沉迷。在这些app的设置中找到个性化推荐，然后关闭。就不会再继续推荐自己感兴趣的内容。也就不会在沉迷了。

hexo要新增文章，可以用“hexo new 文章标题“这个命令，会在source\posts 创建一个md文件，建议用Typora编辑。

Typora官网：https://www.typoraio.cn/（89元终身使用，推荐正版）

破解版：蓝奏云文件

用hexo搭建博客

发表于 2025-01-04

1准备工作

下载nodejs和git,vscode并安装

输入

敲4次Enter

然后进入C:\Users\用户名，在里面进入.ssh文件，用文本编辑器打开，复制ssh keys,打开github,粘贴，如图

然后输入ssh -T git@github.com 如果显示不能连接就输入ssh -T -p 443 git@ssh.github.com

这是因为防火墙或网络设置可能会阻止 SSH 端口 22 的通信，后面一个是443端口，基本不会被封。

在喜欢位置新建文件Blog，然后进入文件夹

右键空白处然后点Git bash here，依次输入

hexo install

hexo g 生成静态文件。
这个命令会将 Markdown 文件转化为 HTML 页面，
hexo s 启动本地服务器，预览博客效果。
在本地启动一个开发服务器，方便在发布前查看博客页面效果。

用记事本打开_config.yml

branch: main

npm install hexo-deployer-git –save
