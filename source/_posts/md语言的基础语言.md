Markdown 是一种轻量级的标记语言，用于快速生成结构化文档。以下是常用语法：

### 1. **标题**

```
markdown复制代码# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

效果：

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

------

### 2. **字体样式**

```
markdown复制代码*斜体* 或 _斜体_
**加粗** 或 __加粗__
***加粗斜体***
~~删除线~~
```

效果：
*斜体*
**加粗**
***加粗斜体***
~~删除线~~

------

### 3. **列表**

- **无序列表**：用 `-`、`*` 或 `+` 表示。

```
markdown复制代码- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
```

- **有序列表**：用数字和 `.` 表示。

```
markdown复制代码1. 第一项
2. 第二项
   1. 第二项的子项
```

效果：

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

1. 第一项
2. 第二项
   1. 第二项的子项

------

```

```

### 6. **表格**

```
markdown复制代码| 表头1 | 表头2 | 表头3 |

| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |
```

| ----- | ----- | ----- |
| 数据1 | 数据2 | 数据3 |

------

### 7. **代码块**

- **单行代码**：用反引号 ` 包裹。

  ```
  markdown
  
  
  复制代码
  `代码`
  ```

  示例：`print("Hello World")`

- **多行代码**：用三个反引号 ``` 包裹。

  ```
  markdown复制代码```python
  def hello():
      print("Hello World")
  ```

  ```
  复制代码
  ```
